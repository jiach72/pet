## Context
社区生态功能是 PetPulse 从"硬件数据展示"升级为"宠物生活操作系统"的关键。这些功能将覆盖宠物主人的全生命周期需求：日常生活服务、购物、公益、社交活动、健康教育、保险保障。

## Goals / Non-Goals

### Goals
- 构建完整的生活服务地图，支持 POI 展示和智能推荐
- 搭建轻量级商城框架（用品销售 + 宠物寄卖）
- 提供社区活动平台（团建、配对、婚礼、葬礼）
- 建立健康科普内容体系
- 接入宠物保险服务

### Non-Goals
- 不实现真实支付系统（商城仅展示 + 跳转）
- 不实现真实地图 SDK（使用 Mock + 系统导航）
- 不实现视频播放器（使用系统播放器）
- 不实现后端服务（全部 Mock 数据）

## Decisions

### 1. 页面结构
```
app/
├── (tabs)/map.tsx          # 地图主页（POI + 推荐）
├── services/               # 生活服务
│   ├── poi-detail.tsx      # POI 详情
│   └── insurance.tsx       # 保险服务
├── shop/                   # 商城
│   ├── index.tsx           # 商城首页
│   └── pet-listing.tsx     # 宠物寄卖
├── community/              # 社区
│   ├── events.tsx          # 活动列表
│   ├── charity.tsx         # 爱心公益
│   ├── wedding.tsx         # 宠物婚礼
│   ├── memorial.tsx        # 宠物纪念
│   └── matching.tsx        # 宠物配对
└── education/              # 科普
    ├── index.tsx           # 科普首页
    └── article.tsx         # 文章详情
```

### 2. 主题色方案
| 模块 | 主色 | 场景 |
|------|------|------|
| 生活服务 | #3B82F6 (蓝) | 信任、专业 |
| 商城 | #F97316 (橙) | 购物、活力 |
| 公益 | #EC4899 (粉) | 爱心、温暖 |
| 婚礼 | #F472B6 (粉红) | 喜庆 |
| 葬礼 | #374151 (灰黑) | 庄重 |
| 科普 | #10B981 (绿) | 健康、成长 |

### 3. 导航入口
- 地图 Tab 作为生活服务入口
- 新增"服务"快捷入口区域
- 科普内容可从首页健康模块进入

## Risks / Trade-offs
| Risk | Mitigation |
|------|------------|
| 功能过多导致开发周期长 | 分阶段实现，优先核心功能 |
| Mock 数据与真实场景差异 | 定义清晰接口，便于后续替换 |
| 婚礼/葬礼功能敏感性 | UI 设计需谨慎，提供隐私保护 |

## Open Questions
- 商城是否需要独立 Tab？（建议：不需要，从服务入口进入）
- 科普内容是否支持 UGC？（建议：初期仅官方内容）
