# Change: 社区生态功能模块 (Community Ecosystem)

## Why
根据 PRD V2.0，PetPulse 需要构建完整的"宠物生活操作系统"。当前已完成守护（体征监测）和社交（情感日记、绿盾、挑战赛）功能，需要补充生活服务、商城、公益、活动、科普、保险等核心功能模块，形成完整闭环生态。

## What Changes

### 🗺️ 生活服务地图 (F-10)
- 宠物医院/商店 POI 展示与导航
- 基于健康数据的精准推荐（体重↑推荐游泳馆，压力高推荐安抚服务）

### 🛒 宠物商城
- 宠物用品销售
- 宠物售卖/寄卖平台

### ❤️ 爱心公益
- 宠物救治众筹
- 宠物赠与/领养平台

### 🎉 社区活动中心 (F-11)
- 宠物团建活动报名
- 宠物配对服务
- **宠物葬礼**（简易/宗教葬礼，庄重 UI）
- **宠物婚礼**（婚礼请柬，喜庆 UI）

### 📚 健康科普
- 《如何看懂宠物的 HRV 报告？》
- 《从心率曲线发现心脏病的早期信号》
- 《宠物的"压力"原来有生理指标！》
- 系列短视频/图文内容

### 🛡️ 宠物保险 (F-12)
- 保险产品展示
- 基于健康分的动态保费计算

## Impact
- Affected specs: `life-services`, `pet-commerce`, `pet-care`, `health-education` (新建)
- Affected code:
  - `app/(tabs)/map.tsx` - 地图页重构
  - 新增 `app/services/` - 生活服务子页面
  - 新增 `app/shop/` - 商城子页面
  - 新增 `app/community/` - 社区活动子页面
  - 新增 `app/education/` - 科普内容子页面
  - 新增 `components/` 相关组件
